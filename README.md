# Python图书信息管理系统 python772

## 项目概述

Python图书信息管理系统是基于Python语言开发的一款信息管理系统，旨在为图书馆或其他图书管理机构提供一个便捷的书籍信息管理解决方案。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- **用户模块**
  - 登录
  - 注册
- **书籍管理模块**
  - 书籍信息管理
  - 书籍排序
  - 书籍搜索
- **信息管理模块**
  - 出版社信息管理
  - 作者信息管理
- **分页查询**

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- Django框架
- MySQL数据库
